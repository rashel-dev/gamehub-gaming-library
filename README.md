# GameHub Game Library

**Live URL**: [https://gamehub-assignemnt.netlify.app/](https://gamehub-assignemnt.netlify.app/)

## Purpose

GameHub is a modern and responsive web application for discovering and managing your favorite video games. It provides a user-friendly interface to browse a vast library of games, view details, and keep track of the ones you love. With user authentication powered by Firebase, you can create a personal account, manage your profile, and maintain a personalized list of favorite games.

## Key Features

*   **User Authentication**: Secure login and registration system using Firebase Authentication.
*   **Game Discovery**: Browse for games from a large collection.
*   **Detailed Game Information**: View comprehensive details for each game.
*   **Favorites List**: Add and manage a personal list of your favorite games.
*   **User Profiles**: View and update your user profile.
*   **Responsive Design**: A seamless experience across all devices, built with Tailwind CSS and DaisyUI.
